Introduccion

Se desarrollara la plataforma "Arrienda tu finca", esta plataforma busca ofrecer la facilidad de arriendar fincas, se junta al arrendador con el arrendatario, intermediando todo el proceso de arriendo, desde la busqueda, la elecciony solicitudes. Para el arrendatario se le brinda toda la libertad de administracion de sus propiedades.

Tecnologias Principales

Para el desarrollo de esta plataforma se hace uso de un amplio stack tecnologico.

  1. Spring boot: Se usa Spring boot, framework de Java, para el desarrollo de la Rest API
  2. PostgreSQL: Se usa PostgreSQL para la base de datos relacional
  3. Angular js: Se usa Angular js para el desarrollo de la pagina web (frontend)
  4. Kubernetes: El despliegue de la plataforma se hace por medio de Kubernetes
